# shalini-shygan

A minimal personal profile site built with React and Vite.

## Setup

```bash
yarn install
```

## Development

```bash
yarn start
```

Opens at [http://localhost:5173](http://localhost:5173)

## Build

```bash
yarn build
```

## Tech

- React 18
- Vite
- Plain CSS (no frameworks)
- Google Fonts (Space Mono + Inter)
